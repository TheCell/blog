# Code Highlighting

Focused `// [!code focus]`
```csharp
public class CustomEvent : UnityEvent<T0> {}
public class CustomEvent : UnityEvent<T0, T1> {}
public class CustomEvent : UnityEvent<T0, T1, T2> {}  // [!code focus]
public class CustomEvent : UnityEvent<T0, T1, T2, T3> {}
``` 

Highlighting: `// [!code highlight]`, `// [!code warning]`, `// [!code error]`

```csharp
// GameObjectEvent.cs
using System;
using UnityEngine; // [!code highlight]
using UnityEngine.Events; // [!code warning]

[Serializable]
public class GameObjectEvent : UnityEvent<GameObject> { } // [!code error]
```

Diff `// [!code --]` and `// [!code ++]`
```csharp
// Health.cs
using UnityEngine;

public class Health : MonoBehaviour
{
    [SerializeField] private int _health = 10; // [!code --]
    [SerializeField] private int _currentHealth = 10; // [!code ++]

    public void SetHealth(int health)
    {
        _health = health; // [!code --]
        _currentHealth = health; // [!code ++]
    }
}
```
